
# AutoClicker

AutoClicker is a simple application for automating mouse clicks at regular intervals. It is designed for Windows only.

## Usage
```

Ahora, aquí tienes la redacción del uso de la aplicación en inglés y en español:

**Uso de la aplicación (en inglés):**

1. Download the latest release of AutoClicker from the [Releases](https://github.com/RoberthDudiver/Clickwherever/releases) page.

2. Extract the downloaded ZIP file to a desired location on your computer.

3. Double-click the `AutoClicker.exe` file to launch the application.

4. Place the cursor on the desired location where you want the clicks to occur.

5. Specify the time interval between clicks in milliseconds using the provided input field.

6. Click the "Start" button to begin the automatic clicking.

7. To stop the clicking, press the "Stop" button or close the application window.

**Uso de la aplicación (en español):**

1. Descarga la última versión de AutoClicker desde la página de [Releases](https://github.com/RoberthDudiver/Clickwherever/releases).

2. Extrae el archivo ZIP descargado en la ubicación deseada de tu computadora.

3. Haz doble clic en el archivo `AutoClicker.exe` para iniciar la aplicación.

4. Coloca el cursor en la ubicación deseada donde quieres que ocurran los clics.

5. Especifica el intervalo de tiempo entre los clics ensegundos utilizando el campo de entrada proporcionado.

6. Haz clic en el botón "Start" para comenzar los clics automáticos.

7. Para detener los clics, presiona el botón "Stop" o cierra la ventana de la aplicación.
